Let's you choose a font for your posts or pages from the admin edit page for your posts or pages.  Will load the styles into the page dynamically as that post or page is displayed.

== Description ==

This plugin allows you to choose a font when editing a post or page, that will effect that post or page on the front-end. You can customize which fonts you want available to choose from on the edit pages.  You can also tell the plugin which elements to apply the styles to, so you are not limited to just the content of the post or page.

== Installation ==

1. Upload post-font-selector directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Make sure that css/fontSelect.css and css/frontEndFonts.css are both 777 so that we can write these when you make changes to your settings.

== Frequently Asked Questions ==

= What is this plugin for? =

To allow you to dynamically modify the fonts that are used on the post or page level. You can also have this plugin only modify a specific part of your post page (this will apply for all posts or all pages).

= How do I apply the font to my pages? =

Identify the HTML element identifier for your content area of your theme. HTML ID is preferred, but class would work alright if it is unique enough. Put this identifier into the admin page in the content HTML identifiers area.
